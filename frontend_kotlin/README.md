# Frontend UMEBOT (App Android en Kotlin)

Este directorio contiene el codigo fuente de la aplicacion cliente que se instala en la tablet del robot Pepper.

## Arquitectura

La aplicacion sigue una arquitectura **MVVM (Model-View-ViewModel)** para una clara separacion de responsabilidades y un codigo mantenible.

## Configuracion y Compilacion

1.  Abre el directorio `frontend_kotlin` en **Android Studio**.
2.  El IDE detectara automaticamente la configuracion de Gradle. Espera a que se sincronicen las dependencias.
3.  Para compilar un APK de depuracion, puedes usar la opcion del menu `Build > Build Bundle(s) / APK(s) > Build APK(s)`.
